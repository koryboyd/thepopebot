This single skill consolidates several agent utilities:

### Modify Self
Provides commands and documentation for the agent to update its own source files, configuration, or behavior. Use with caution; changes are tracked via git.

### Browser Automation
Includes helpers for controlling a headless browser, taking screenshots, crawling pages, and extracting text.

### LLM Secrets
Securely manage and access API keys and credentials required by the agent. Keeps secrets out of chat logs and persisted storage.
